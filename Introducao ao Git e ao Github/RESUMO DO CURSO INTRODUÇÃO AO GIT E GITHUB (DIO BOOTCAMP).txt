INTRODUÇÃO AO GIT E GITHUB (DIO BOOTCAMP)
::::::::::::::::::::::::::::::::::::::::

Nesse curso tive uma introducao para utilizacao do (Git e o Github)

Oque é o Git?: Git é um sistema de controle de versões de arquivos

Oque é o GitHub?: GitHub é um servidor baseado em nuvem local onde hospeda as versões de arquivos do Git

Utilizei alguns comandos interessantes para criar, manusear pastas e arquivos do git para o github

::::::::::::::SÃO ELES:::::::::::::::::::

1-Comando para criacao de pasta: (mkdir)
2-Comando para introducao de pasta: (cd)
3-Comando para voltar da pasta: (cd ..)
4-Comando para visualizar os status do git: (git status)
5-Comando para listar arquivos: (dir) ou (ls)
6-Comando para listar arquivos ocultos: (ls-a)
7-Comando para comentário: (git commit -m "")
8-Comando para deletar arquivos: (del)
9-Comando para remoção de repertório: (rmdir)
10-Comando para criação de commit: (git commit)
11-Comando para visualizar alterações feitas: (git log)
12-Comando para enviar commit para repositorio remoto: (git push)
